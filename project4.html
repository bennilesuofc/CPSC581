Group Work Planner

Demo Video

This is a program for organizing and allocating tasks when working in a group (it can also be used for individual work). Tasks and group members are created, and then the tasks are assigned to each group member. Tasks are given a brief description and a due date, and can be marked as completed. Each member can organize their assigned tasks as they see fit by dragging them into a new position in their list. Completed or extra tasks and group members can be removed as desired. In this prototype, there is no limit to how many tasks or group members are created, but any changes made are not saved and do not persist after the site is closed.

An alternate implementation that I considered was to create tasks and then assign group members to each task- this approach would let multiple people be assigned to a single task. Another idea I explored was giving each task a level of importance or urgency, (for example, using text, icons, or different colours), so that more important tasks could be focused on. I decided that letting each group member reorder their list of tasks was sufficient for this prototype
